# Judging Criteria

Projects are judged on:

1. **Shipped application** — is it actually live and working?
2. **Data security** — how well is user data protected?
3. **Execution and demo clarity** — quality of the build and the pitch.

> [!tip] Guiding principle
> **Prioritize creativity over complexity.**
