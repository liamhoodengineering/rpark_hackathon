# The Prompt

> Build something that **helps people**.

## Core Requirement
- **Live web application** — reachable by anyone with an internet connection.

## Pick Three Features
Choose exactly **three** of the following to build into the app:

- [ ] Insight engine: Build charts, maps, dashboards that surface insights from data
- [ ] For you: recommended user content (data personalization)
- [ ] Plugged in: integrate external APIs
- [ ] Who are you: multiple user roles with different positions
- [ ] Where are you: location-based feature
- [ ] Mob mentality: let users contribute & consume shared data
- [ ] Ring ring: send notifs and alerts
- [ ] Make no mistakes: make no mistakes (AI/LLM)
- [ ] Memory lane: Persist state across sessions for user actions, preferences, history
- [ ] Live updates: deliver real-time updates without manual refresh
- [ ] Connected flow: guide users through multi-step workflows based on prior actions
- [ ] Activity feed: track and log system actions publicly over time

See [[Feature Ideas]] to brainstorm which three to commit to.
